# AI Joke Generator

AI Joke Generator using n8n and Groq LLaMA 3.3 - Fetches random jokes via API and translates them to Hindi using AI Agent workflow

## How it works
1. Trigger → HTTP Request fetches random joke
2. AI Agent translates joke to Hindi using Groq LLaMA 3.3
3. Also generates original Hindi dad joke!

## Tools Used
- n8n
- Groq API (LLaMA 3.3)
- Official Joke API
